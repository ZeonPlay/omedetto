Panduan CTF

Untuk menyelesaikan challenge ini:

  1. Cek metadata gambar bocchi.png yang kamu terima
  2. Dekripsi password yang ditemukan
  3. Kunjungi URL hasil dekripsi (website ini)
  4. Inspect element untuk petunjuk selanjutnya
1
Enkripsi URL di Terminal

Gunakan terminal untuk meng-enkripsi URL website ini:

Enkripsi Base64 + ROT13

$ echo -n "https://yourwebsite.com" | base64

# Hasil Base64: aHR0cHM6Ly95b3Vyd2Vic2l0ZS5jb20=


$ echo "aHR0cHM6Ly95b3Vyd2Vic2l0ZS5jb20=" | tr 'A-Za-z' 'N-ZA-Mn-za-m'

# Hasil ROT13: aUGgHCHfN6YyFe91H9jR6HB/F92dXVi9yYg==


# String terakhir adalah password untuk metadata gambar

2
Tambahkan ke Metadata Gambar

Gunakan exiftool untuk menyisipkan password:

Tambahkan Metadata

$ exiftool -Comment="aUGgHCHfN6YyFe91H9jR6HB/F92dXVi9yYg==" bocchi.png

# Password berhasil ditambahkan ke field Comment


$ exiftool -ImageDescription="CTF Password: aUGgHCHfN6YyFe91H9jR6HB/F92dXVi9yYg==" bocchi.png

# Atau tambahkan ke field ImageDescription

3
Verifikasi Metadata

Cek apakah password sudah tersimpan:

Baca Metadata

$ exiftool bocchi.png | grep -E "(Comment|ImageDescription)"

# Comment: aUGgHCHfN6YyFe91H9jR6HB/F92dXVi9yYg==

# ImageDescription: CTF Password: aUGgHCHfN6YyFe91H9jR6HB/F92dXVi9yYg==
